Si votre ordinateur a déjà les modules Pygame, sys, os, time, PyQt5 installés, 
vous pouvez exécuter les fichiers .py.
Sinon exécutez les fichiers .exe et exécutez les niveaux suivants à la fin de chaque niveau.

Les fichiers ne sont pas exécutables depuis un logiciel de code, il faut les exécuter via un CMD ou avec un 
double-click.
